# Metal Tracker
## Introduction
Are you playing in an eco position but have the memory of a goldfish?
Perhaps a dirty freeloading teammate is insisting they have paid, but you think they might be telling porkies*?
Or maybe it's just seeing statistics give you a dopamine hit? 
     *Just give T2 for free anyway, strong teams win games!

Metal Tracker can help you out!
This simple widget lets you see in real time who has sent you what, so you never need to forget to fulfil a contract again.
Also works for spectators.
(Un)limted offer, get now and track not only metal, but energy too!

# Use Instructions
Instruction on use are in the widget - display can be toggled on and off (default Alt+J).
Compatible with uikeys.txt - see widget instructions.
In game settings to resize / move widget.
