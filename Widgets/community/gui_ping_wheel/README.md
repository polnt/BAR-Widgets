# Ping Wheel Widget for Beyond All Reason

This is originally [zxbc's ping wheel widget](https://github.com/zxbc/BAR_widgets). As zxbc was no longer able to keep working on it, I was granted permission to create this repo so the community could continue improving it and bug fixing for current version of BAR. This version fixes a bug that was introduced after the game was updated, and also customizes the default widget messages.

# Usage
Hold down a keybind (default Alt+F) a ping wheel will be displayed at your mouse cursor. Clicking onto one of the options will then place a ping with the appropriate label on the map.
